# Summary

This is the current working code for my Version 3 Portal Gun. It implements a state machine with a default tick-rate of 200Hz. It controls two speakers (Idle-Humm and SFX), the LEDs and detects a (currently unused) Rotary Encoder as a future input method.

Right now I sadly don't have the time to make proper instructions but you can take the older Version as reference.
